Общие правила оформления задач
========

В рамках второго семестра требуется выполнить 10 задач. Каждая задача должна быть оформлена в виде отдельного task{i}.ipynb файла и tensorboard{i}.zip файла.
В каждом файле *.ipynb должно быть:

- построение архитектуры;
- выполнен процесс обучения;
- показан пример работы модели до обучения и после;

Файл .zip должен содержать результаты эксперимента в формате tensorboard для каждой из задач:

- для каждого набора параметров свой график зависимости качества от обучения (если требуется в задаче);
- примеры работы модели в процессе обучения модели.

Для каждой задачи должны быть представлены выводы:

- какой результат ожидали;
- какой не ожидали;
- что было не ясно.

Код и эксперимент должен быть понятным внешнему читателю:

- В коде должны быть коментарии;
- Названия переменных должно быть интерпретируемые.

P.S. Рекомендуется все вычисления проводить на google colab в режиме cuda.

P.S.S. Рекомендуется использовать backup моделей при обучении на google drive.


Правила сдачи
========

Всего на семестр выдано 10 задач, они разделены на 4 блока:


Блок 1. Глубокие нейронные сети (CNN, RNN, LSTM)
Задача 1. Анализ модели CNN
Задача 2. Анализ модели LSTM

Блок 2. Обучение без учителя и векторные представления текстов
Задача 3. Модель автокодировщика
Задача 4. Вариационный автокодировщик
Задача 5. Распознавание именованных сущностей на основе fasttext

Блок 3. Внимание, трансформеры, рекомендательные системы
Задача 6. Генерация аннотации к изображению
Задача 7. Детекция машинной генерации текста
Задача 8. Рекомендательная система на основе матричного разложения

Блок 4. Тематическое моделирование, обучение с подкреплением, рекомендательные системы
Задача 9. Классификация даты документа на основе тематического моделирования
Задача 10. Posterior Sampling в обучении с подкреплением


На каждый блок даётся по 3 недели сдач*.
Блок 1 Сдавать можно до 13 Марта
Блок 2 Сдавать можно до 3 Апреля
Блок 3 Сдавать можно до 24 Апреля
Блок 4 Сдавать можно до 8 Мая (*тут меньше на 1 неделю, чтобы на зачётку не попасть)

- Блоки можно сдавать в любые сдачи до дедлайна. На сдаче сдаём блок только целиком, нельзя по частям. Если не можете к сдаче решить часть задач из блока, то все нерешённые зануляются. (не относится к уже прошедшим сдачам)
- Если решение оказывается сгенерированным нейронкой и Вы не можете его достаточно пояснить, то за задачу ставится максимум 50% баллов. При полном отсутствии понимания решения ставится 0 без возмонжости перерешать задание. Полная субъективность преподавателя, решение не оспаривается и основывается в том числе на истории взаимодействия со студентом.
